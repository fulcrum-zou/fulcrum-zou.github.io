我用到的环境是:

  • macOS Catalina 10.15.7
  • Visual Studio Code 1.59.0

2 设置环境变量

打开或新建~/.bash_profile.

我的配置是

  • GOPATH：日常开发的根目录. 1.8版本之前必须指定该变量，且不能与GOROOT(安装目录)相同；1.8版本之后GOPATH有默认值，可以依据情况自己更改.
  • PATH：环境变量. 需要将GOBIN目录加到PATH路径下.
